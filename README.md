Hidden Texas History Project

The Hidden Texas History project started purely to find a picture for a
family member. It's become more than that though; with a goal to help
folks unlock the true beauty of Texas in everyone here and past.

Of course, this may seem strange to have a github repository for a
history project, but the fact is that AI is so much better and faster
at doing things that we're going to exploit it to unlock the past
that's been hidden in plain sight.

The very first addition here is an AutoHotkey v2 script to assist in
automating the capture, digitization, preservation, and availability
of the monumental work done by so many Texans over the last 200+
years.
